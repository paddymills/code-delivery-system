
# Code Delivery System frontend

written in [svelte-js](https://svelte.dev/) using [bootstrap](https://getbootstrap.com/) components via [sveltestrap](https://sveltestrap.js.org/)

## Development

Install 

```bash
cd svelte
yarn

# or use npm (tauri scripts use yarn)
npm install
```

start tauri app
```bash
cargo tauri dev
```
